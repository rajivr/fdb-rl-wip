fn take_box(&mut self) -> *mut dyn Any + Send

ðŸ”¬This is a nightly-only experimental API. (std_internals)

Take full ownership of the contents. The return type is actually Box<dyn Any + Send>, but we cannot use Box in core.

After this method got called, only some dummy default value is left in self. Calling this method twice, or calling get after calling this method, is an error.

The argument is borrowed because the panic runtime (__rust_start_panic) only gets a borrowed dyn BoxMeUp.
