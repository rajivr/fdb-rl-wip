Struct ion_rs::reader::UserReader

pub struct UserReader<R: RawReader> { /* private fields */ }
Expand description

A streaming Ion reader that resolves symbol IDs into their corresponding text.

Reader itself is format-agnostic; all format-specific logic is handled by the wrapped RawReader implementation.
