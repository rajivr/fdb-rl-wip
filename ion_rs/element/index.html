Module ion_rs::element

source ·
Expand description

Provides a dynamically typed, materialized representation of an Ion value.

An Element represents an (annotations, value) pair, where a value is an Ion integer, float, list, struct, etc.

For reference here are a couple other value style APIs for JSON:

  • simd_json’s Value API
  • serde_json’s Value API
