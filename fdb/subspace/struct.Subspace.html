Struct fdb::subspace::Subspace

source · 
pub struct Subspace { /* private fields */ }
Expand description

Subspace provides a convenient way to use Tuple to define namespaces for different categories of data.

Get the key encoding of the specified Tuple in this Subspace for use with SetVersionstampedKey.

Panic

The index where incomplete versionstamp is located is a 32-bit little-endian integer. If the generated index overflows u32, then this function panics.

Panic
